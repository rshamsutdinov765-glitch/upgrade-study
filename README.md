# 🚀 UpGrade Study Club

Современный образовательный лендинг для записи учеников на курсы.

## 📚 О проекте

Сайт создан для учебного центра **UpGrade Study Club**.
Основная цель — привлечение учеников и удобная запись через WhatsApp.

---

## 🌐 Страницы сайта

* `index.html` — главная страница
* `math.html` — курс Математики
* `english.html` — курс Английского
* `speedreading.html` — курс Скорочтения
* `aftercare.html` — Продлёнка

---

## ✨ Функционал

* 💎 Современный дизайн (уровень лендингов)
* 🔝 Фиксированное меню (navbar)
* ✨ Плавные анимации при скролле
* 📱 Полная адаптивность (телефон / планшет)
* 💬 Запись через WhatsApp
* 🔗 Переходы между страницами
* 🚀 Быстрая загрузка (чистый HTML + CSS + JS)

---

## 🛠 Технологии

* HTML5
* CSS3 (анимации, градиенты, flex/grid)
* JavaScript (обработка формы, UI эффекты)
